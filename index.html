Frequently Asked Questions

How Feather Wand helps performance engineers?

Feather Wand helps performance engineers by providing a way to generate JMeter scripts with ease. It uses Anthropic/OpenAI API to generate the scripts and elements.

Can I customize the prompts?

Yes, you can customize the prompts to your needs.

Which programming languages are supported?

Feather Wand supports Java and Groovy primarily. But it can be customized using the prompts.

Can I use Feather Wand offline?

No, it requires an internet connection to use the API.

How much it actually costs?

It's free to use. You just need to provide your own Anthropic/OpenAI API key. The cost of the API is covered by you based on your usage.
